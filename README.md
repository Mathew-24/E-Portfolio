# Mathew Venzon Portfolio Website

## Project overview

This repository contains the source code for Mathew Venzon's portfolio website.

## How can I edit this code?

You can clone this repository locally and use your preferred IDE to make changes.

1. Clone the repo:
   ```sh
   git clone <YOUR_GIT_URL>
   cd <YOUR_PROJECT_NAME>
   ```
2. Install dependencies and start the development server:
   ```sh
   npm install
   npm run dev
   ```

This will launch the site locally with hot reloading.

The only requirement is having Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

Follow these steps:

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd <YOUR_PROJECT_NAME>

# Step 3: Install the necessary dependencies.
npm i

# Step 4: Start the development server with auto-reloading and an instant preview.
npm run dev
```

You can also edit files directly on GitHub or use GitHub Codespaces if you prefer cloud-based development environments.

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## How can I deploy this project?

Deploy the build output (`npm run build`) to any static hosting provider such as Netlify, Vercel, GitHub Pages, or your own server.
